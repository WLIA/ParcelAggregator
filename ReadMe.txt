Purpose
-------

Script adds a set of common fields to GIS parcel polygon layers, than transfers existing attribute data into new schema.
It then appends the processing parcel data to a aggregate parcel layer for say an entire state or region.

Requirements/Setup
-------------

* ArcGIS 10 to access arcpy modules  

1. Open the Parcel_Tools.tbx and set the source for the python script tool to the location of AggregateParcels.py.  
2. Run the Script tool from ArcGIS client.
